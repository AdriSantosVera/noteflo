# Configuración de herramientas de inteligencia artificial

## Entorno de trabajo

El desarrollo de NoteFlow se apoya en un flujo de trabajo asistido por inteligencia artificial, con el objetivo de mejorar la productividad, mantener la coherencia del proyecto y optimizar la toma de decisiones técnicas.

Las herramientas principales utilizadas son:

- Visual Studio Code como entorno de desarrollo.
- Codex como asistente de programación integrado en el editor.
- ChatGPT como apoyo para planificación, documentación y resolución de dudas.

---

## Visual Studio Code

Visual Studio Code se utiliza como entorno principal de desarrollo. Permite editar archivos, navegar por la base de código y mantener una estructura organizada del proyecto.

Además, facilita la integración con herramientas de inteligencia artificial y otras extensiones que mejoran la productividad durante el desarrollo.

---

## Codex

Codex actúa como asistente de programación dentro del entorno de desarrollo, permitiendo acelerar la implementación y mantener una mayor coherencia en el código.

Se emplea principalmente para:

- Crear y organizar la estructura inicial del proyecto.
- Generar archivos y componentes base.
- Refactorizar código y mejorar la organización.
- Mantener consistencia en la arquitectura.
- Automatizar tareas repetitivas dentro del desarrollo.

---

## ChatGPT

ChatGPT se utiliza como herramienta complementaria enfocada en la toma de decisiones y la mejora del proyecto a nivel conceptual.

Sus principales usos son:

- Definir la arquitectura del proyecto.
- Redactar y mejorar la documentación.
- Resolver dudas técnicas y conceptuales.
- Analizar decisiones de producto.
- Apoyar el desarrollo en tareas que no requieren edición directa del código.

---

## Justificación del uso de IA

La combinación de estas herramientas permite mantener un equilibrio entre automatización y control del desarrollo. Mientras Codex facilita la generación y modificación del código, ChatGPT aporta una visión más estratégica, ayudando a estructurar el proyecto y a tomar decisiones con mayor claridad.

Este enfoque contribuye a mejorar la calidad del resultado final y a optimizar el flujo de trabajo durante todo el desarrollo de NoteFlow.
